1. p : p is used to write a paragraph in html document . p is block level element

  2. H: The heading tag is used in HTML to define headings of a page. Headings are defined by (hn) , with “n” being a number between 1 and 6, and determines which position a heading has in the hierarchy of the heading structure. Heading tags have a particularly important function within an HTML structure.

  3. Div : . The div tag defines a division or a section in an HTML document. The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The divtag is easily styled by using the class or id attribute.

  4. Span : The span tag is an inline container used to mark up a part of a text, or a part of a document. The span tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute. The span tag is much like the div element, but div is a block-level element and span is an inline element.
  5. A: A tag is used to define hyperlink in html doc.hyperlink means if you click redirect other page.
  6. Img: image tag is used to add image in web page.
